# Bitehub (React App)

A responsive food delivery web application UI built using React and Tailwind CSS.
This project focuses on food browsing experience, including filtering, categories, search functionality, and price display.

# Features

- Search food items by name
- Filter food by categories (Burger, Pizza, Drinks, etc.)
- Display prices for all food items
- Call-to-Action (CTA) buttons (UI only)
- Fully responsive design (mobile, tablet, desktop)
- Reusable and modular React components

# TechStack

- React JS – Component-based frontend
- Tailwind CSS – Utility-first styling
- JavaScript (ES6+)
- Vite – Fast development environment
- Git and Github
- CSS

# Installation & Setup

1. Clone the repository


2. Move into the project folder
- cd bitehub

3. Install dependencies
npm install

4. Run the development server
npm run dev

- Creating reusable React components

- Implementing UI logic using props and hooks

- Building responsive layouts with Tailwind CSS
